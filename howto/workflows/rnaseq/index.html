RNAseq Analyses

Allows processing of RNAseq data using…

  • STAR-htseq
  • PRADA

Setup

  • Download and extract GLASS tools tarball and docker image on a docker on a compatible compute node with at least 200 GB disk space, 64GB RAM and 24 VCPUs

  • Setup docker image and disk mounts

To-do on client side

  • Organize sample wise RNAseq fastq files under /mnt/disk2/glassdata/inputs/fqs/sampleX/

Dry Run

  • Helper script: disk1/glasscore/configs/bin/glass_rnaseq which has similar argument structure as for DNA alignment workflow with following new or updated arguments:

    • -a: filename(s) and not full path of read 1 of sample fastq file
    • -b: filename(s) and not full path of read 2 of sample fastq file
    • -m: Run mode can be either STAR or PRADA
# cd to code repository
cd /path_to_disk1/glasscore/configs/bin/

## Run STAR-htseq workflow
./glass_rnaseq -m STAR -a sampleX_read1.fq -b sampleX_read2.fq -s rnaseq_star_sampleX -t DRY | tee -a run.log

## Run PRADA workflow
./glass_rnaseq -m PRADA -a sampleX_read1.fq -b sampleX_read2.fq -s rnaseq_star_sampleX -t DRY | tee -a run.log

change -t DRY to -t GO
