Phase 1 data release

Following GLASS Face2Face meetup at SNO 2018 meeting, we have now migrated to an updated and stable version of analysis workflows. To access updated workflows and the first release of mutation and clinical annotation dataset, you will need a prior approval to access https://synapse.org/glass data portal. Please contact Prof. Roel Verhaak for additional details.

This website now hosts a legacy version of workflows which were containerized in docker/singularity. Until we containerizing updated snakemake based workflows in containers, documentation on this website should be considered outdated.
