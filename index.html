This website hosts how-to manuals for running various analysis pipelines for the GLASS consortium.

Preprint

We are moving to Singularity based workflows for better compatibility with academic IT infrastructure. You can download beta built of Singularity image at https://github.com/glass-consortium/glasstools
Both Docker and Singularity images are near identical with a few changes in ENTRYPOINT and CMD specifications. While docker and Singularity images are stable enough to execute workflows, documentation and run scripts (shell wrappers) to run workflows are not ready yet for production release. All of run scripts reside outside of image and (when available) are sourced by image entrypoint/cmd specifications at execution of workflow. We will soon update documentation to run workflows.

Please report issues related to workflows at https://github.com/glass-consortium/glassdocs/issues and Singularity image at https://github.com/glass-consortium/glasstools/issues
